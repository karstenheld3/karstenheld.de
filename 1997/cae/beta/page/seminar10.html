Seminar 9


Integration von Ingenieuranwendungen

In diesem Seminar wird an einem Beispiel die Verwendung von Visual Basic for Applications und der Automation-Technologie gezeigt. Es wird ein Vorgang automatisiert, in welchem drei typische Ingenieuranwendungen (AutoCAD R14, Excel 97, Beamax) Verwendung finden.

Visual Basic for Applications

Visual Basic für Applikationen (VBA) ersetzt die herkömmlichen Makroprogrammiersprachen der Office-Produkte und soll zukünftig in allen Anwendungen zur Verfügung stehen. Sie soll den Grundstein für die Integration und Konsistenz sämtlicher unterstützender Anwendungen bilden. VBA ist eine Erweiterung von Visual Basic 3.0. und vereint drei Hauptmerkmale:
  • VBA verfügt über eine Entwicklungsumgebung mit Editier- und Testhilfen, Dialogeditor, Online-Dokumentation und Makrorecorder.
  • VBA erlaubt Zugriff auf APIs und somit die Verwendung der Systemfunktionalität.
  • VBA ermöglicht die Verwendung von OLE und erlaubt die direkte Steuerung einzelner Objekte mit Hilfe der OLE-Automation
