IMAGO:

A music related project, that was formed in '94 by Lutz Röder and Karsten Koch. Results until now are the IMAGO Orpheus (a track-based music sequencer), IMAGO Wave (an instrument/sample editor) and IMAGO Audio Interface (a C-library for developers).

IMAGO Advanced Computer Music:

A comercial music project. We compose music for multimedia applications and computer games. Test our demo-musics and listen to all the archieved soundtracks (most files in S3M-format). For contact information click here.

Computer Aided Engineering (CAE):

This page contains a lot of engineering related stuff, that mostly was done for the Lehrstuhl für Informatik im Bauwesen at the Bauhaus Universität Weimar. Durchlaufträger is a program that demonstrates the use of Microsoft Foundation Classes (MFC). Beamax is a redesigned version of the Durchlaufträger example. Femax is the current project.
